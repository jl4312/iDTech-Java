# Arithmetic Operators

**Addition**: `+`

**Subtraction**: `-`

**Multiplication**: `*`

**Division**: `/`

**Remainder (Modulo)**: `%`

**Increment By 1**: `++`

**Decrement By 1**: `--`

See the [code](https://github.com/RamV13/iDTech-Java/blob/master/Lesson%203/Code.java) for the syntax of using these operators on numbers.
